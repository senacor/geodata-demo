= AngularJS Prototype

== Dev Setup
* first install npm...

```
$ npm install -g grunt-cli
$ npm install -g bower
$ npm install --global yo
$ npm install --global generator-angular
$ npm install compass
$ gem install compass
$ yo angular
$ grunt serve
```

== Dependencies

* declared in package.json


== YoMan Stuff
